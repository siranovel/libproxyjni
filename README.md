# libproxyjni #
c言語からjavaプログラムを呼び出すライブラです。

## Description ##
c言語からJNI(Java Native Interface)を利用して、Javapログラム中のメゾットを呼び出すインターフェースを提供です。
c言語で、オブジェクト指向風に作成しています。
詳細は、https://github.com/siranovel/libproxyjni/wiki
